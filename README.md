# בוט טלגרם - תורים בכללית ללא סיסמא

בוט שמחפש אוטומטית תורים פנויים לרופא מקצועי בכללית (שירות "תור ללא סיסמא").

---

## שלב 1 — יצירת בוט טלגרם וקבלת API Key

1. פתח טלגרם וחפש את המשתמש **@BotFather**
2. שלח לו את הפקודה: `/newbot`
3. תן שם לבוט (למשל: `Clalit Appointment Bot`)
4. תן username לבוט — חייב להסתיים ב-`bot` (למשל: `clalit_appoint_bot`)
5. BotFather ישלח לך **HTTP API Token** שנראה כך:
   ```
   1234567890:ABCDefGhIJKlmNoPQRsTUVwxyZ
   ```

---

## שלב 2 — הכנת הפרויקט

### דרישות מקדימות
- Python 3.8 ומעלה
- Chrome מותקן במחשב

### התקנה

```bash
# שכפל / הורד את הקבצים לתיקייה
cd BotTelegram

# התקן תלויות
pip install -r requirements.txt
```

### שמירת ה-API Key

צור קובץ בשם `key.txt` בתיקיית הפרויקט והדבק בו את הטוקן שקיבלת מ-BotFather:

```
key.txt
───────
1234567890:ABCDefGhIJKlmNoPQRsTUVwxyZ
```

> **חשוב:** אל תעלה את `key.txt` ל-GitHub! הוסף אותו ל-`.gitignore`

---

## שלב 3 — הפעלת הבוט

```bash
python general.py
```

הבוט יתחיל לרוץ — תראה אין פלט מיוחד, זה תקין.

---

## שלב 4 — שימוש בבוט

פתח את הבוט בטלגרם ושלח `/start`.

**תהליך השיחה:**

| שלב | מה הבוט שואל | מה להזין |
|-----|--------------|----------|
| 1 | מספר זהות | 9 ספרות (למשל: `123456789`) |
| 2 | שנת לידה | 4 ספרות (למשל: `1990`) |
| 3 | תאריך יעד | DD.MM.YYYY (למשל: `31.12.2025`) |
| 4 | התמחות | לחץ על כפתור מהרשימה |

לאחר בחירת ההתמחות — הבוט מתחיל לחפש אוטומטית.  
אם לא נמצא תור לפני התאריך שהזנת, הוא ינסה שוב **כל 10 דקות**.

**התמחויות נתמכות:**
- אורטופדיה
- אף אוזן גרון
- חירורג שד
- נשים
- עור
- עיניים

## מבנה הקבצים

```
BotTelegram/
├── general.py          # הבוט הראשי
├── automation_fill.py  # Selenium — גולש באתר כללית
├── requirements.txt    # תלויות Python
├── key.txt            # ← אתה יוצר קובץ זה! (לא נמצא ב-repo)
└── README.md
```

---

## פתרון בעיות נפוצות

| בעיה | פתרון |
|------|-------|
| `FileNotFoundError: key.txt` | צור את קובץ `key.txt` עם הטוקן |
| Chrome לא נמצא | ודא ש-Google Chrome מותקן |
| שגיאת `selenium` | הרץ שוב `pip install -r requirements.txt` |
| הבוט לא מגיב | בדוק שה-terminal עדיין פעיל עם `python general.py` |
| הבוט נמצא תור אבל לא לפני התאריך | נסה תאריך יעד רחוק יותר |
